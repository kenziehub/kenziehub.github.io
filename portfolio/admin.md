# Admin

This page was used in the static HTML version of the site as a lightweight browser editor for `data/content.js`.

## Current Behavior

- password-gated in the browser
- loads current site content
- exports `content.js` or `content.json`
- intended for manual GitHub Pages updates

## Note For Hugo

This page is probably not needed in the Hugo rebuild unless you plan to keep a custom browser-based editing workflow.
